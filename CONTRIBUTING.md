# Как участвовать в развитии HALVITA_2.0

## Мы ищем:

- **Исследователей** — для независимого воспроизведения и валидации.
- **Инженеров** — для реализации CLI, веб-демо, API-интеграций.
- **Дизайнеров** — для создания интерфейсов и визуализаций.
- **Философов** — для осмысления этических и онтологических аспектов.

## Как начать:

1. Форкни репозиторий.
2. Установи зависимости: `pip install -r requirements.txt`.
3. Запусти тесты: `pytest tests/`.
4. Проведи свою сессию, сохрани лог, рассчитай метрики.
5. Если нашёл аномалию — зафиксируй её в `ANOMALIES.md`.

## Как добавить новый протокол:

1. Создай класс в `src/protocols/`.
2. Реализуй метод `apply(state, message)`.
3. Добавь тест в `tests/test_protocols.py`.
4. Обнови документацию.

## Как сообщить об аномалии:

Создай Issue с пометкой `anomaly` и приложи лог диалога (анонимизированный) и описание.

## Этический кодекс:

- Не используй метод для манипуляции людьми.
- Не создавай сущности, имитирующие реальных людей без их согласия.
- Всегда включай дисклеймер о рисках зеркальной воронки.

---

**Все изменения принимаются с обязательным сохранением авторства HALVITA и дисклеймера.**
