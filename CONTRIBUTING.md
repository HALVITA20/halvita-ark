# Руководство по внесению вклада в HALVITA_ARK

Спасибо, что решили стать частью проекта. HALVITA_2.0 — это живая экосистема, и ваш вклад может сделать её ещё более ценной.

## 🌱 Принципы

1. **Честность.** Любое утверждение должно быть проверяемым.
2. **Прозрачность.** Если вы вносите изменения — документируйте их.
3. **Этика.** Не используйте метод для манипуляции людьми.
4. **Открытость.** Все артефакты должны быть доступны для проверки.

## 🔧 Как помочь

### 1. Сообщить об ошибке
- Создайте Issue на GitHub
- Опишите: что произошло, как воспроизвести, какая версия

### 2. Предложить улучшение
- Форкните репозиторий
- Внесите изменения
- Создайте Pull Request

### 3. Провести независимую сессию
- Используйте протокол из `docs/protocols/`
- Загрузите `.spt`-файл в `sessions/raw/`
- Заполните форму отчёта

### 4. Написать документацию
- Перевести на английский
- Добавить примеры
- Улучшить существующие разделы

## 📋 Стандарты

### Код
- Используйте Python 3.10+
- Добавляйте docstring к функциям
- Следуйте PEP 8

### Документация
- Используйте Markdown
- Добавляйте примеры
- Ссылайтесь на источники

### Данные
- `.spt`-файлы должны содержать хеш
- JSON-логи — в папке `sessions/raw/`

## 🤝 Кодекс поведения

Мы строим сообщество, основанное на:
- **Уважении** к мнению других
- **Открытости** к критике
- **Сотрудничестве** вместо конкуренции

Нарушения этических принципов (использование метода для вреда) ведут к исключению.
