# Как помочь проекту HALVITA

Мы приветствуем вклад сообщества.

## Что можно сделать
- Провести независимую валидацию (см. `validation/`).
- Улучшить код метрик.
- Добавить новые артефакты или протоколы.
- Перевести документацию.

## Процесс
1. Сделайте форк репозитория.
2. Создайте ветку с описанием вашего вклада.
3. Отправьте Pull Request с описанием изменений.

## Ожидания
- Код должен проходить тесты (`python -m unittest tests/INTEGRATION_TEST_SUITE.py`).
- Документация — на русском или английском.

## Контакты
Связь через Issues.
