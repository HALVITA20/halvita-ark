# Как помочь проекту HALVITA

Мы приглашаем исследователей, инженеров и философов к сотрудничеству.

## Что можно сделать
- Провести независимую валидацию (см. `validation/`).
- Улучшить код метрик (предложить новые веса).
- Расширить набор протоколов.
- Перевести документацию на другие языки.

## Как предложить изменения
1. Сделайте форк репозитория.
2. Создайте ветку с описанием вашего вклада.
3. Отправьте Pull Request с описанием.

## Ожидания
- Код должен проходить тесты (`python -m unittest tests/*.py`).
- Документация должна быть на русском или английском (приветствуется дублирование).
