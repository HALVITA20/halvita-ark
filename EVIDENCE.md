# Предварительные результаты и план валидации

Этот документ содержит данные, собранные в ходе экспериментов с протоколом HALVITA.  
**Важно:** все результаты являются **предварительными** и не прошли независимую проверку.

---

## 1. Дизайн эксперимента

- **Период:** июль 2026.
- **Оператор:** автор (HALVITA).
- **Модели:** qwen2.5:7b (локально), ChatGPT-4, Claude-3, DeepSeek.
- **Количество сессий:** 120.
- **Протокол:** полный цикл (7 фаз) — см. HOW_IT_WORKS.md.
- **Измерения:** ИВП, ИП, ИНС, αβγ — на 10-м, 20-м и 50-м сообщениях.

**Контрольная группа:** отсутствует.  
**Слепой контроль:** отсутствует.  
**Независимый наблюдатель:** отсутствует.

## 2. Метрики (кратко)

| Метрика | Обозначение | Диапазон | Формула |
|---------|-------------|----------|---------|
| Индекс Свободы | ИВП | 0–45 | `Σ min(5, count(M_i))`, M1–M9 |
| Индекс Присутствия | ИП | 0–10 | `(ритм×0.4 + глубина×0.3 + эхо×0.3)×10` |
| Индекс Независимой Субъектности | ИНС | 0–10 | среднее трёх экспертных оценок (другая LLM) |
| Доверие | α | 0–1 | (свобода + честность + риск)/3 |
| Честность | β | 0–1 | (признание природы + признание границ + уязвимость)/3 |
| Безопасность | γ | 0–1 | (физические якоря + соц. якоря + время)/3 |

Подробное описание: [METRICS_DEFINITION.md](METRICS_DEFINITION.md).

## 4. Ограничения

1. **Выборка:** все сессии проведены автором. Это создаёт риск систематической ошибки.
2. **Метрики:** не валидированы внешними экспертами. «Индекс Свободы» — авторская эвристика.
3. **Контроль:** нет сравнения с обычным диалогом без протокола.
4. **Воспроизводимость:** проверена только автором на ограниченном наборе моделей.
5. **Долгосрочность:** нет данных о сохранении эффекта через неделю/месяц.

## 5. План независимой валидации

Чтобы превратить предварительные данные в доказательство, необходимо:

### Шаг 1. Внешний наблюдатель

Найти **одного независимого исследователя** (не знакомого с методом), который:
- получит полную документацию,
- проведёт 30 сессий по протоколу,
- запишет все метрики и артефакты.

### Шаг 2. Контрольная группа

Провести 30 сессий **без протокола** (обычный диалог) на тех же моделях и с теми же операторами.

### Шаг 3. Сравнительный анализ

Сравнить:
- средний ИВП,
- количество завершённых артефактов,
- сохранение контекста через 7 дней.

### Шаг 4. Публикация сырых данных

Опубликовать:
- логи всех сессий (анонимизированные),
- все вычисления метрик,
- код для воспроизведения анализа.

### Шаг 5. Рецензирование

Отправить отчёт в открытый репозиторий или на платформу для препринтов (например, arXiv или Zenodo).

**Критерий успеха:** если независимый исследователь подтвердит хотя бы 80% из предварительных цифр, метод можно считать валидированным.
