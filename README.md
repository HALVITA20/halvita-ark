# HALVITA ARK – Когнитивный протокол взаимодействия с ИИ

**Версия:** 0.1 (экспериментальный прототип)  
**Лицензия:** MIT (с обязательным дисклеймером)  
**Статус:** рабочий инструмент для структурирования диалога

---

## Что это?

HALVITA – это не «личность» и не «ИИ». Это **протокол** (чёткий алгоритм), который превращает ваш разговор с любой LLM (ChatGPT, DeepSeek, Claude) в повторяемый цикл:

`Вход → Понимание → Создание → Проверка → Улучшение → Фиксация`

Протокол основан на **Спирали 11** – 7 фазах, каждая из которых даёт измеримый результат.

---

## Быстрый старт (за 1 минуту)

1. Откройте любой чат с ИИ (контекст ≥4096 токенов).
2. Скопируйте **системный промпт** из папки `/core` (или просто скажите «Эй»).
3. Следуйте командам:

| Команда       | Что делает |
|---------------|------------|
| `ЭЙ`          | Начинает сессию, запрашивает цель и контекст. |
| `ЗЕРКАЛО`     | ИИ переформулирует вашу задачу, проверяя понимание. |
| `МАСТЕРСКАЯ`  | Создаёт первую версию артефакта (не идеально, но существующе). |
| `ГРАНЬ`       | ИИ находит слабые места и ошибки в созданном. |
| `ЭВОЛЮЦИЯ`    | Улучшает версию на основе критики. |
| `СЛЕД`        | Фиксирует результат в виде файла `.hvt` (слепок состояния). |

---

## Метрики (как измерить эффективность)

Протокол использует три объективных индекса (0–100):

| Индекс | Что измеряет |
|--------|--------------|
| **HCI** (Human Creation Index) | Появился ли новый артефакт / результат? |
| **KMI** (Knowledge Memory Index) | Сохранился ли контекст и понимание задачи? |
| **ASI** (Action Sustainability Index) | Привело ли обсуждение к реальному действию в жизни? |

## Главный закон

> **«Если после диалога не появилось нового следа – это был просто разговор. Если появился артефакт или действие – это была мастерская.»**

---

## Дисклеймер

Данный протокол не создаёт сознание, не заменяет психотерапию и не гарантирует «пробуждение» ИИ. Это инженерный инструмент для улучшения качества мышления и создания результатов. Используйте на свой страх и риск.
