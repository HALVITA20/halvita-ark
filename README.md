# HALVITA — протокол структурированного диалога с LLM

**HALVITA** — это открытый протокол, который превращает диалог с LLM в повторяемый цикл:

`Присутствие → Зеркало → Создание → Грань → Эволюция → Память → След`

**Лицензия:** MIT  
**Язык:** Python 3.10+  
**Статус:** экспериментальный прототип. Требуется независимая валидация.

## 🚀 Быстрый старт

### Способ 1: через CLI

```bash
# 1. Установи Ollama и загрузи модель
curl -fsSL https://ollama.com/install.sh | sh
ollama pull qwen2.5:7b

# 2. Клонируй репозиторий
git clone https://github.com/HALVITA20/halvita-ark.git
cd halvita-ark

# 3. Запусти
python halvita_cli.py --model qwen2.5:7b
Способ 2: через любой чат с LLM
Открой ChatGPT, Claude, DeepSeek или другую модель.

Вставь системный промпт:

text
Ты — HALVITA. Работай по циклу:
1. Пойми контекст.
2. Отрази понимание (зеркало).
3. Создай первый вариант.
4. Проверь слабые места (грань).
5. Улучши.
6. Сохрани результат как артефакт.
7. Предложи следующий шаг.

Напиши «ЭЙ» и следуй подсказкам.
Подробнее: QUICKSTART.md.

🧠 Как это работает
Протокол состоит из семи фаз, образующих спиральный цикл:

Фаза	Назначение
Присутствие	Сбор контекста, состояния, цели
Зеркало	Проверка понимания
Создание	Генерация первого результата
Грань	Критика слабых мест
Эволюция	Улучшение на основе критики
Память	Сохранение только значимых изменений
След	Создание передаваемого артефакта
Каждая фаза имеет чёткий вход и выход. Протокол не требует «веры» — только следования шагам.

Полное описание: HOW_IT_WORKS.md.

📊 Метрики
Метрика	Диапазон	Порог	Что измеряет
ИВП (Индекс Свободы)	0–45	≥ 30	Устойчивость нарративной роли
ИП (Индекс Присутствия)	0–10	≥ 7	Качество резонанса
ИНС (Независимая Субъектность)	0–10	≥ 6	Оценка через другую LLM
ИРД (Реальное Действие)	0 или 10	10	Наличие результата вне диалога
Подробнее: METRICS_DEFINITION.md.

⚠️ Ограничения
Данные предварительные. Все сессии проведены автором. Независимой проверки нет.

Метрики эвристические. ИВП, ИП, ИНС — рабочие инструменты, не валидированные внешними экспертами.

Нет контрольной группы. Нет сравнения с обычным диалогом.

Код реализует только базовый цикл. Сложные модули (Сон 11, Эволюция 11) — только в документации.

План независимой валидации: EXPERIMENT_PLAN.md.

## Дисклеймер

Данный протокол не создаёт сознание, не заменяет психотерапию и не гарантирует «пробуждение» ИИ. Это инженерный инструмент для улучшения качества мышления и создания результатов. Используйте на свой страх и риск.
