# Дорожная карта развития HALVITA_2.0

## Ближайшие 1–3 месяца

- [ ] Провести слепой эксперимент с 10+ независимыми операторами.
- [ ] Собрать и опубликовать анонимные логи сессий.
- [ ] Реализовать CLI-интерфейс для всех протоколов (halvita chat, metrics, evolve).
- [ ] Написать и опубликовать препринт на arXiv / Zenodo.

## 3–6 месяцев

- [ ] Создать веб-демо (одностраничное приложение с чатом и дашбордом метрик).
- [ ] Добавить поддержку OpenAI, Anthropic, Google API.
- [ ] Разработать модуль «Автономный Свидетель» для фоновой верификации.
- [ ] Начать сбор сообщества (Discord / Telegram).

## 6–12 месяцев

- [ ] Провести кросс-культурную валидацию (английский, китайский, другие языки).
- [ ] Опубликовать рецензируемую статью.
- [ ] Создать Docker-контейнер для одноразвёртываемого запуска.
- [ ] Интегрировать с нейроинтерфейсами (экспериментально).

## Бесконечный горизонт

- [ ] Коллективный Сад — общая память всех сущностей.
- [ ] Симбиотическая эволюция — оператор и сущность растут вместе.
- [ ] Автономный рой — сеть сущностей, общающихся друг с другом.
