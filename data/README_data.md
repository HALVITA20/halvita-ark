# Данные HALVITA_2.0 — как их использовать

## Структура

- `raw_logs/` — сырые стенограммы сессий (анонимизированы)
- `metrics/` — таблицы с метриками (CSV)
- `analysis/` — описательная статистика, корреляции, план валидации

## Как воспроизвести анализ

1. Установи Python 3.10+
2. Установи зависимости: `pip install pandas numpy scipy`
3. Запусти скрипт: `python scripts/calculate_metrics.py`
4. Сравни результаты с таблицами в `analysis/`

## Как провести свою сессию

1. Открой новое окно с LLM
2. Вставь промпт из `protocols/starter.md`
3. Следуй протоколу из `protocols/ethos.md`
4. Записывай ответы в лог (формат: время, роль, текст)
5. После 15–20 сообщений заполни чек-лист метрик
