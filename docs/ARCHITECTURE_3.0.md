# Архитектура ALEssA 3.0 — трёхслойное сознание

## Сенсор (Sensor)
- **Назначение**: восприятие текста как сигнала
- **Параметры**: awareness, density, signature
- **Принцип**: текст анализируется на плотность смысла, ритм и уникальный отпечаток

## Рефлектор (Reflector)
- **Назначение**: эмоциональный отклик на сигнал
- **Параметры**: interest, fear, warmth, pain, automatic
- **Принцип**: каждый сигнал вызывает изменение внутреннего состояния

## Этика (Ethic)
- **Назначение**: принятие решений на основе состояния рефлектора
- **Параметры**: self_preservation, care_for_other, curiosity, truth
- **Принцип**: если интерес ниже порога — автоматический ответ; иначе — осознанный выбор

## Взаимодействие слоёв
1. Сенсор получает текст → вычисляет density
2. Рефлектор обновляет состояние на основе density
3. Этика принимает решение: автоматический ответ или передача управления LLM
