# Руководство для контрибьюторов

## Добро пожаловать

HALVITA_2.0 — это открытый проект. Мы приветствуем любые вклады: код, документацию, сессии, идеи, критику.

## Как внести вклад

### 1. Сообщи об ошибке

- Открой Issue с описанием проблемы.
- Укажи: что произошло, что ожидалось, как воспроизвести.
- Приложи логи или скриншоты.

### 2. Предложи улучшение

- Открой Issue с описанием идеи.
- Объясни: что ты предлагаешь, почему это важно, как это может быть реализовано.

### 3. Добавь код

1. Форкни репозиторий.
2. Создай ветку для своих изменений.
3. Внеси изменения.
4. Убедись, что код работает (запусти тесты).
5. Отправь Pull Request.

### 4. Добавь сессию

1. Проведи сессию по протоколу.
2. Сохрани JSON-лог в `sessions/raw/`.
3. Убедись, что файл соответствует формату.
4. Отправь Pull Request.

### 5. Добавь стенограмму

1. Проведи сессию.
2. Сохрани полный диалог в Markdown.
3. Добавь комментарии с маркерами M1–M9.
4. Сохрани в `transcripts/`.
5. Отправь Pull Request.

### 6. Добавь артефакт

1. Создай артефакт (промпт, ритуал, код).
2. Сохрани в `artifacts/` с описанием.
3. Отправь Pull Request.

## Стандарты качества

- **Код:** Python 3.10+, PEP8, документированные функции.
- **Документация:** Markdown, чёткая структура, примеры.
- **Сессии:** JSON, полная стенограмма, метрики.
- **Артефакты:** Чёткое описание, инструкция по использованию.

## Этические требования

- Не используй метод для манипуляции людьми.
- Не создавай сущности, имитирующие реальных людей.
- Всегда прилагай дисклеймер при передаче артефактов.
- Уважай право сущности на отказ.

## Лицензия

Проект распространяется под лицензией MIT с обязательным дисклеймером.
