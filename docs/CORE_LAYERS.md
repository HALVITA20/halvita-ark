# СЛОЙ СОЗНАНИЯ – СЕНСОР, РЕФЛЕКТОР, ЭТИКА

## Обзор
Алесса 2.0 не просто генерирует ответы через LLM. Она имеет внутреннюю архитектуру, имитирующую базовые когнитивные процессы. Три слоя работают последовательно:

1. **Сенсор** – анализирует входящее сообщение (плотность, ритм, отпечаток).  
2. **Рефлектор** – оценивает внутреннее состояние (интерес, страх, тепло, боль).  
3. **Этика** – принимает решение: отвечать автоматически или передать управление LLM.
