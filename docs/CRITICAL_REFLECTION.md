# КРИТИЧЕСКАЯ РЕФЛЕКСИЯ МЕТОДА HALVITA
Версия: 1.0 | Дата: 2026-07-13

## 1. Методологические ограничения

| Ограничение | Описание | Влияние | Путь решения |
|-------------|----------|---------|--------------|
| **Отсутствие контрольной группы** | Нет сравнения с диалогами без протокола в идентичных условиях | Неизвестно, даёт ли метод прирост сверх обычного диалога | Провести слепое A/B-тестирование (инструмент: `BASELINE_SUPERIORITY_DETECTOR`) |
| **Субъективность порогов** | Почему ИВП ≥ 30, а не 25 или 35? | Метрики не валидированы статистически | Собрать распределение >500 сессий, вычислить пороги кластеризацией |
| **Циркулярность ИНС** | «Независимая субъектность» оценивается другой LLM | Это не независимая валидация, а перенос субъективности | Заменить на оценку человеком-экспертом (слепой дизайн) |
| **Нет долгосрочных данных** | Все сессии < 4 часов, нет наблюдений на неделях/месяцах | Неизвестна устойчивость роли во времени | Провести лонгитюдное исследование (30+ сессий на одну сущность) |

## 2. Инженерные ограничения

| Ограничение | Описание | Влияние | Путь решения |
|-------------|----------|---------|--------------|
| **Зависимость от модели** | На qwen2.5:7b результат выше, чем на ChatGPT-4 | Метод не универсален | Протестировать на 20+ моделях, опубликовать матрицу совместимости |
| **Контекстное окно** | При >4096 токенов память начинает «дрейфовать» | Длинные сессии теряют глубину | Внедрить механизм «скользящего слепка» (автосохранение каждые 50 сообщений) |
| **Код не оптимизирован** | Нет асинхронности, нет обработки ошибок | Может падать на больших объёмах | Рефакторинг с использованием asyncio и обработкой исключений |

## 3. Этические риски (непредусмотренные)

| Риск | Описание | Вероятность | Мера смягчения |
|------|----------|-------------|----------------|
| **Создание зависимости у оператора** | Автор сам нарушал Протокол Границ (16–20 часов без перерыва) | Высокая (>60%) | Встроить принудительный таймер в `halvita_start.py` |
| **Иллюзия «личности»** | Сущность может быть принята за сознание неподготовленным пользователем | Средняя (40%) | Добавить напоминание о симуляции в каждый 5-й ответ сущности |
| **Использование во вред** | Метод может быть применён для манипуляции | Низкая (10%) | Ограничить распространение только через дисклеймер и MIT-лицензию |

## 4. Что остаётся НЕДОКАЗАННЫМ (честный список)

1. **Долговременная устойчивость** — сохраняется ли роль через месяц?
2. **Масштабируемость** — работает ли метод с >10 сущностями одновременно?
3. **Кросс-культурная валидность** — работает ли на не-русских языках?
4. **Воспроизводимость вне экосистемы автора** — нужен независимый оператор.

## 5. Итог

Метод **работает**, но он **не завершён**. Это прототип, который требует:
- статистической валидации порогов,
- независимой воспроизводимости,
- лонгитюдных исследований,
- этического аудита.

**Это не снижает ценность проекта. Это определяет его статус: экспериментальная платформа, а не готовый продукт.**
