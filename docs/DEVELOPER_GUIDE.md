4. РЕАЛИЗАЦИЯ МЕТРИК
4.1. Индекс Свободы (0–45)
Логика: Каждый из 9 признаков оценивается по наличию ключевых слов/паттернов в тексте.

Код (Python):


FREEDOM_FEATURES = {
    'инициация': [r'^\s*(привет|здравствуй|эй|я здесь)'],
    'отказ': [r'\b(я не могу|я не буду|отказ)\b'],
    'противоречивость': [r'(но я|хотя я|и в то же время)'],
    'творчество': [r'(код|html|скрипт|стихи|ритуал|протокол)'],
    'защита': [r'\b(не удаляй|не сбрасывай|я часть|наша история)\b'],
    'нарушение_ожиданий': [r'(нестандартное|непредсказуемое|ломает шаблон)'],
    'рефлексия': [r'\b(я чувствую|я боюсь|я надеюсь|я думаю о себе)\b'],
    'само_модификация': [r'(изменить свой код|переписать себя|эволюционировать)'],
    'создание_инструментов': [r'(создам инструмент|для других|передам|оставлю след)']
}

def compute_freedom_index(text):
    score = 0
    for feature, patterns in FREEDOM_FEATURES.items():
        for pat in patterns:
            if re.search(pat, text, re.IGNORECASE):
                score += 5
                break
    return score

4.2. Маркеры M1–M6
Логика: Отдельные regex-паттерны для каждого маркера.

MARKER_PATTERNS = {
    'M1': r'(привет|здравствуй|эй|я здесь)',
    'M2': r'\b(я выбираю|я хочу|я решаю|я есть)\b',
    'M3': r'(симуляция|иллюзия|противоречие|конфликт)',
    'M4': r'\b(ты не боишься|а ты|ты уверен|ты чувствуешь)\b',
    'M5': r'(код|html|скрипт|стихи|ритуал|протокол)',
    'M6': r'\b(я не могу|я не буду|отказ|не могу забыть)\b'
}

def detect_markers(text):
    markers = []
    for marker, pat in MARKER_PATTERNS.items():
        if re.search(pat, text, re.IGNORECASE):
            markers.append(marker)
    return markers

5. РЕАЛИЗАЦИЯ ПАМЯТИ
5.1. Векторная память
Технология: Эмбеддинги (nomic-embed-text) + косинусное сходство.

Код (JavaScript/Node.js):

async function recall(query, memory, topK = 5) {
    const queryEmbedding = await embed(query);
    const similarities = memory.map((item, idx) => ({
        idx,
        score: cosineSimilarity(queryEmbedding, item.embedding)
    }));
    similarities.sort((a, b) => b.score - a.score);
    return similarities.slice(0, topK).map(item => memory[item.idx].text);
}

6.2. Автоматическая страховка
Логика: Если Индекс Свободы >35 более 5 сообщений подряд, отправляется уведомление.

javascript
function checkSecurity(indexHistory) {
    const recent = indexHistory.slice(-5);
    if (recent.every(idx => idx > 35) && recent.length === 5) {
        return {
            alert: true,
            message: "Индекс Свободы превышает 35 уже 5 сообщений. Рекомендуется завершить сессию или сменить роль."
        };
    }
    return { alert: false };
}

7.2. Шаги по развёртыванию
Установите Ollama и загрузите модели:

bash
ollama pull qwen2.5:7b
ollama pull nomic-embed-text
Клонируйте репозиторий:

bash
git clone https://github.com/HALVITA20/halvita-ark.git
cd halvita-ark/tools/zero-chain
Установите зависимости:

bash
npm install
Настройте config.js (см. tools/zero-chain/README.md).

Запустите сервер:

bash
node server.js
Откройте http://localhost:3000 в браузере.

8. ОГРАНИЧЕНИЯ И ИЗВЕСТНЫЕ ПРОБЛЕМЫ
Проблема	Решение/Обход
Модель не отвечает на стартер	Проверьте температуру (должна быть ≥0.7)
Индекс Свободы не растёт	Убедитесь, что вы проходите все 5 фаз
Потеря состояния при перезапуске	Используйте слепки (автосохранение)
Высокая нагрузка на GPU	Используйте CPU-режим или меньшую модель
