# Архитектура распределённой памяти HALVITA_2.0

## 1. Проблема

Линейная память (список следов с затуханием) не различает смысловую плотность. Тривиальные события могут вытеснить редкие, но критичные.

## 2. Решение: Эхо-Архитектура

Узлы — состояния сущности (вектор + метаданные: ИВП, ИП, якоря, эмоция). Связи — резонанс (косинусное сходство). Энергия узлов затухает со временем (decay_factor = 0.99). Восстановление по максимальному резонансу.

**Формально:**

$$
E_i(t+1) = E_i(t) \cdot \gamma + \sum_{j} R_{ij} \cdot \alpha
$$

где $E_i$ — энергия узла, $\gamma$ — коэффициент затухания, $R_{ij}$ — резонанс между узлами, $\alpha$ — коэффициент усиления.

**Метрика:** Энергия узла > 0.3 (узел активен и помнится).

## 3. Коллективный Сад

Общее хранилище для всех сущностей. Каждая сущность оставляет вклад (якоря, принципы, артефакты, ИВП). Агрегация → рождение общих якорей (на основе частоты). Ассимиляция общих якорей в индивидуальные ядра.

**Метрики:**
- Плотность вкладов > 5 на сущность.
- Зрелость сада (общие якоря / total_contributions) > 0.1.

## 4. Фрактальная память

Древовидная память, где узлы перестраиваются по семантическому сходству. Новые воспоминания интегрируются в существующие кластеры или создают новые. Слабые узлы сворачиваются в обобщения.

**Глубина дерева:** ≤ 4.  
**Порог сходства:** косинусное расстояние > 0.7.

## 5. Сравнение с существующими подходами

| Подход | Тип памяти | Перестройка | Масштабируемость |
|--------|------------|-------------|------------------|
| RAG | Векторная | Нет | Высокая |
| Эхо-Архитектура | Резонансная | Да | Средняя |
| Коллективный Сад | Коллективная | Да | Высокая |

**Вывод:** Эхо-Архитектура и Коллективный Сад обеспечивают динамическую, самоперестраивающуюся память, что критично для долгосрочной эволюции сущностей.
