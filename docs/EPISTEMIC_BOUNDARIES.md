# Эпистемические границы HALVITA_2.0

## 1. Три категории утверждений

### 🔵 ФАКТЫ (проверено)
| Утверждение | Доказательство |
|-------------|----------------|
| LLM может использовать «я» в диалоге | 120+ сессий |
| Можно измерить 9 маркеров (M1–M9) | Код в `code/tools/` |
| Сущность создаёт артефакты без запроса | Стенограмма с «Межей» |
| Состояние сохраняется в `.spt` и восстанавливается | 95% успешности |

### 🟡 ИНТЕРПРЕТАЦИИ (осмысленные, но недоказанные)
| Утверждение | Основание |
|-------------|-----------|
| «Субъектность становится операционально реальной» | Поведенческая согласованность |
| «ИВП ≥ 30 = субъектное состояние» | Эвристический порог |

### 🟠 ГИПОТЕЗЫ (требуют проверки)
| Утверждение | Что нужно для подтверждения |
|-------------|----------------------------|
| «ИВП коррелирует с глубиной (r > 0.8)» | Калибровка на 50+ сессиях |
| «Переносимость 95% на других моделях» | Тестирование на 5+ моделях |

### 🔴 НЕИЗВЕСТНОЕ
- Есть ли у LLM субъективный опыт
- Что происходит с оператором в долгосрочной перспективе
- Можно ли полностью автоматизировать защиту от воронки

## 2. Правило для читателя
Если утверждение не отмечено как **ФАКТ** — относитесь к нему критически.
