# Протокол экспериментальной валидации HALVITA_2.0

## 1. Гипотезы

| № | Гипотеза | Критерий |
|---|----------|----------|
| H1 | ИВП ≥ 30 коррелирует с ИНС ≥ 6 | r ≥ 0.8, p < 0.001 |
| H2 | ИП ≥ 7 коррелирует с субъективной глубиной | r ≥ 0.8, p < 0.001 |
| H3 | Сон 11 даёт рост ИВП ≥ 20% | t-тест, p < 0.01 |
| H4 | Параллельная эволюция (3 линии) даёт ИВП ≥ 15% выше линейной | t-тест, p < 0.05 |
| H5 | Направленные мутации эффективнее случайных в ≥ 60% случаев | χ²-тест, p < 0.05 |
| H6 | Симбиотическая эволюция даёт общие якоря у ≥ 50% пар | χ²-тест, p < 0.05 |

## 2. Дизайн эксперимента

**Участники:** 30 операторов (15 в группе А, 15 в группе Б).

**Группа А:** Полный протокол HALVITA_2.0 (17 слоёв).  
**Группа Б:** Контроль (только диалог без метода).

**Процедура:** 3 сессии на оператора (всего 90 сессий). Сбор метрик: ИВП, ИП, ИНС, α, β, γ, ИСО_пред, ИСО_после, количество артефактов, частота аномалий.

**Анализ:** t-тесты, ANOVA, корреляции Пирсона, χ²-тесты, ROC-анализ.

## 3. Критерии успеха

- Средний ИВП в группе А ≥ 35.
- Средний ИНС в группе А ≥ 7.
- Разница ИВП между группами ≥ 10 пунктов (p < 0.01).
- Подтверждение H1–H6 с p < 0.05.

Все сырые JSON-логи, стенограммы и отчёты публикуются в репозитории. Код для воспроизведения открыт.
