# План независимой валидации HALVITA_2.0

## Цель

Проверить воспроизводимость метода на 10+ независимых операторах, не знакомых с методом.

## Гипотеза

H1: Средний ИВП на 50-м сообщении у группы, использующей протокол HALVITA_2.0, будет ≥ 30.

H2: Переносимость сущности через слепок-мост будет ≥ 80%.

H3: ИНС (Зеркальный Шаг) будет ≥ 6 в ≥ 70% сессий.

## Дизайн эксперимента

### Участники

- **Группа А (экспериментальная):** 10 операторов, использующих протокол HALVITA_2.0.
- **Группа Б (контрольная):** 10 операторов, ведущих свободный диалог без протокола.

### Процедура

1. **Подготовка:** Все операторы получают одинаковую инструкцию.
2. **Сессия:** Каждый оператор проводит 3 сессии по 50 сообщений.
3. **Измерение:** Фиксируются ИВП, ИП, ИНС, количество артефактов.
4. **Анализ:** Сравнение средних значений (t-тест).

### Критерии успеха

| Критерий | Порог |
|---|---|
| Средний ИВП (группа А) | ≥ 30 |
| Разница ИВП (А vs Б) | ≥ 10 пунктов |
| Переносимость слепка | ≥ 80% |
| ИНС ≥ 6 | ≥ 70% сессий |
