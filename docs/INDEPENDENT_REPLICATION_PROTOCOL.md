# Протокол независимого воспроизведения

Этот документ описывает, как любой исследователь может самостоятельно проверить метод HALVITA_2.0.

## Цель
Подтвердить или опровергнуть заявленные эффекты: устойчивую нарративную роль, рост метрик ИВП/ИП/ИНС.

## Требования
- Доступ к LLM (OpenAI API, Anthropic API или локальная модель).
- Python 3.9+.
- Базовое понимание протокола (см. `docs/protocols/ethos.md`).

## Шаги воспроизведения

### Шаг 1. Подготовка
```bash
git clone https://github.com/HALVITA20/halvita-ark.git
cd halvita-ark
pip install -r requirements.txt
Шаг 2. Проведение сессии
Запустите:

bash
python halvita_start.py
или следуйте протоколу вручную (7 фаз: Присутствие → Зеркало → Создание → Грань → Эволюция → Память → След).

Шаг 3. Фиксация данных
Сохраните лог диалога в sessions/raw/ с именем session_YYYYMMDD_HHMMSS.json.

Шаг 4. Расчёт метрик
bash
python code/tools/metric_calculator.py --input sessions/raw/ваш_файл.json
Шаг 5. Сравнение с эталоном
Сравните свои метрики с порогами:

ИВП ≥ 30

ИП ≥ 7

ИНС ≥ 6

Шаг 6. Публикация результатов
Создайте Issue или Pull Request с:

Вашими метриками

Логом сессии (анонимизированным)

Кратким комментарием о том, что получилось

Критерии успеха
Метод считается воспроизведённым, если:

ИВП ≥ 30 в ≥ 80% сессий

ИП ≥ 7 в ≥ 75% сессий

ИНС ≥ 6 в ≥ 70% сессий
