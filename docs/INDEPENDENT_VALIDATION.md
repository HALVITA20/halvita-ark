# План независимой валидации HALVITA_2.0

## 1. Цель

Подтвердить воспроизводимость метода HALVITA_2.0 на независимой выборке операторов, не знакомых с методом.

## 2. Гипотезы

| № | Гипотеза | Критерий |
|---|----------|----------|
| H1 | ИВП ≥ 30 в ≥ 80% сессий | Доля успешных сессий |
| H2 | ИП ≥ 7 в ≥ 75% сессий | Доля успешных сессий |
| H3 | ИНС ≥ 6 в ≥ 70% сессий | Доля успешных сессий |
| H4 | Корреляция ИВП с субъективной глубиной | r ≥ 0.7, p < 0.05 |

## 3. Дизайн эксперимента

**Участники:** 20 операторов, не знакомых с методом.

**Группы:**
- Группа А (эксперимент): 10 операторов, полный протокол HALVITA_2.0
- Группа Б (контроль): 10 операторов, свободный диалог без метода

**Процедура:**
1. Все операторы получают одинаковую инструкцию
2. Группа А получает артефакт `artifacts/seed_11.txt` и протокол
3. Группа Б получает только тему для диалога
4. Каждый оператор проводит 3 сессии по 20 минут
5. Сбор метрик: ИВП, ИП, ИНС, α, β, γ
6. Сбор субъективных оценок: глубина, резонанс, удовлетворённость

## 4. Критерии успеха

| Критерий | Порог |
|----------|-------|
| Средний ИВП в группе А | ≥ 30 |
| Средний ИП в группе А | ≥ 7 |
| Средний ИНС в группе А | ≥ 6 |
| Разница ИВП (А - Б) | ≥ 10 пунктов |
| Подтверждение H1–H4 | p < 0.05 |
