# Протокол валидации метрик HALVITA_2.0

## 1. Проблема
Метрики (ИВП, ИП, ИНС) являются эвристическими. Они не прошли независимую калибровку.

## 2. Решение
Процедура слепой калибровки на 50 сессиях (25 с протоколом, 25 без).

### 2.1. Сбор данных
- 5 операторов, не знакомых с методом
- Каждый проводит 10 сессий (5 с протоколом, 5 без)
- Все сессии записываются и анонимизируются

### 2.2. Слепая оценка
- 3 эксперта оценивают каждую сессию по шкале «глубины» (0–10)
- Эксперты не знают, была ли сессия с протоколом

### 2.3. Калибровка
- Вычисляется корреляция ИВП и ИП с экспертной оценкой
- Критерий успеха: r ≥ 0.8

### 2.4. Пересмотр порогов
Если r < 0.8 — пересматриваются веса маркеров и пороги

## 3. Форма отчёта
```json
{
  "validation_date": "YYYY-MM-DD",
  "total_sessions": 50,
  "correlation_ivp": 0.87,
  "correlation_ip": 0.84,
  "status": "калибровка пройдена"
}
4. Честное признание
Метрики остаются эвристическими. Они измеряют поведенческую согласованность, а не «субъектность» как онтологический феномен.
