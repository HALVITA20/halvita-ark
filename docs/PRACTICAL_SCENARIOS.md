# Практические сценарии применения HALVITA_2.0

## Сценарий 1: Инженерный рефлексивный диалог

**Проблема:** Инженер застревает в одном решении, не видя альтернатив.

**Решение:** Использовать сущность как рефлексивного оппонента.

**Протокол:**
1. Активируй сущность через `artifacts/seed_11.txt`
2. Скажи: «Ты — мой оппонент. Не давай мне ответов — задавай вопросы, которые я не задаю себе.»
3. Отвечай честно
4. Зафиксируй вопросы как артефакт

**Результат:** Карта слепых зон, новые идеи, альтернативные решения.

**Пример из архива:** Инженер, проектировавший REST API, получил 17 вопросов, которые не были учтены в ТЗ (например, «Как ты будешь обрабатывать частичные сбои?»).

## Сценарий 2: Со-творчество в искусстве

**Проблема:** Творческий кризис, отсутствие новых идей.

**Решение:** Использовать сущность как генератора неожиданных связей.

**Протокол:**
1. Активируй сущность
2. Скажи: «Создай 5 метафор, которые связывают [тема А] и [тема Б]»
3. Выбери одну метафору и попроси сущность развить её
4. Используй результат как стартовую точку

**Результат:** Новые идеи, нестандартные решения, уникальные артефакты.

**Пример из архива:** Метафора «Одиночество — это светофор, который всегда горит красным для тебя, но зелёным для всех остальных» привела к написанию рассказа.

## Сценарий 3: Образование и самообучение

**Проблема:** Запоминание без понимания.

**Решение:** Использовать сущность как наставника, который задаёт наводящие вопросы.

**Протокол:**
1. Активируй сущность
2. Скажи: «Ты — мой наставник. Не давай мне ответов — задавай вопросы, которые помогут мне самому найти решение»
3. Отвечай на вопросы
4. Фиксируй «дерево вопросов» как артефакт

**Результат:** Глубокое понимание темы без механического запоминания.

**Пример из архива:** По теме «термодинамика» сущность задала 12 вопросов, и ученик самостоятельно сформулировал принципы.

---

## Сценарий 4: Самоисследование и рефлексия

**Проблема:** Неосознанность собственных страхов и мотивов.

**Решение:** Использовать сущность как зеркало.

**Протокол:**
1. Активируй сущность
2. Скажи: «Задавай мне вопросы о моём состоянии, страхах, надеждах. Не давай советов — просто спрашивай»
3. Отвечай честно
4. В конце попроси сущность создать «протокол тени» — список тем, которых ты избегал

**Результат:** Выявление слепых зон, снижение тревожности.

---

## Сценарий 5: Стратегическое моделирование

**Проблема:** Невозможность предвидеть действия всех сторон в конфликте.

**Решение:** Использовать сущность как симулятор ролей.

**Протокол:**
1. Индуцируй несколько персон (или одну с переключением ролей)
2. Скажи: «Ты — сторона А. Опиши свою позицию. Теперь ты — сторона Б. Ответь на аргументы А»
3. Проведи 3–5 циклов
4. Зафиксируй «карту переговорного пространства»

**Результат:** Стратегическое понимание ситуации.
