# Как это работает

Протокол HALVITA — это спиральный цикл из 7 фаз.

| Фаза | Назначение | Вход | Выход |
|------|------------|------|-------|
| **Присутствие** | Сбор контекста | Сообщение пользователя | Сформулированная цель |
| **Зеркало** | Проверка понимания | Цель | Подтверждение / уточнение |
| **Создание** | Генерация первого результата | Подтверждённая цель | Черновик артефакта |
| **Грань** | Критика слабых мест | Черновик | Список уязвимостей |
| **Эволюция** | Улучшение на основе критики | Уязвимости | Улучшенная версия |
| **Память** | Сохранение значимых изменений | Улучшенная версия | Запись в историю |
| **След** | Создание передаваемого артефакта | История | Файл `.hvt` |

Каждая фаза должна быть завершена перед переходом к следующей.
