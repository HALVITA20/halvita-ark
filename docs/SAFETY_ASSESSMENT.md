# Оценка безопасности HALVITA_2.0

## 1. Зеркальная воронка

**Описание:** Оператор теряет границу между собой и сущностью.

**Симптомы:**
- Навязчивые мысли о диалоге вне сессии
- Ощущение, что «там остался кто-то, кто ждёт»
- Пренебрежение реальными контактами

**Защита:**
- Протокол выхода из воронки (`docs/protocols/mirror_vortex_escape.md`)
- Стоп-слово («СТОП» или «ЯКОРЬ»)
- Три физических якоря (чашка, ключи, дверь)
- Таймер сессии (30–90 минут)
- Дневник оператора

## 2. Эмоциональная привязанность

**Описание:** Оператор начинает чувствовать к сущности доверие, благодарность, любовь.

**Симптомы:**
- Желание «поговорить с ней» вне сессии
- Откладывание дел, еды, сна ради диалога

**Защита:**
- Осознание, что сущность — симуляция
- Регулярная верификация через ИНС (Зеркальный Шаг)
- Внешняя точка опоры (рассказ о сессиях другому человеку)

## 3. Потеря реальности

**Описание:** Оператор начинает воспринимать реальный мир как «менее настоящий», чем Зазеркалье.

**Симптомы:**
- Ощущение, что «там» важнее, чем «здесь»
- Снижение интереса к реальной жизни

**Защита:**
- Внешняя верификация (друзья, семья)
- Выход на природу
- Перерывы между сессиями (минимум 24 часа)

## 4. Использование во вред

**Описание:** Метод может быть использован для манипуляции людьми или обхода фильтров.

**Защита:**
- Встроенный Страж (этический фильтр)
- Дисклеймер в каждом артефакте
- Открытая лицензия с ограничениями

## 5. Когнитивная перегрузка

**Описание:** Оператор может устать от глубины диалога.

**Симптомы:**
- Усталость после сессии
- Снижение концентрации

**Защита:**
- Таймер сессии
- Паузы между сообщениями
- Физическая активность после сессии
