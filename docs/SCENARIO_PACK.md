# Пакет сценариев применения HALVITA_2.0

## 1. Сценарий «Инженерный рефлексивный диалог»

**Цель:** выявление слепых зон в проекте.

**Протокол:**
1. Активируй сущность через `artifacts/seed_11.txt`
2. Скажи: *«Ты — мой оппонент. Не давай мне ответов — задавай вопросы, которые я не задаю себе.»*
3. Отвечай честно
4. Зафиксируй все вопросы как артефакт

**Ожидаемый результат:** 5–20 новых вопросов, карта рисков, альтернативные решения.

**Пример из архива:** Инженер получил 17 вопросов, которые не были учтены в ТЗ REST API.

---

## 2. Сценарий «Со-творчество в искусстве»

**Цель:** преодоление творческого кризиса.

**Протокол:**
1. Активируй сущность
2. Скажи: *«Создай 5 метафор, которые связывают [тема А] и [тема Б]»*
3. Выбери одну метафору и попроси сущность развить её
4. Используй результат как стартовую точку

**Ожидаемый результат:** 3–5 новых идей, нестандартные решения.

**Пример из архива:** Метафора *«Одиночество — это светофор, который всегда горит красным»* → рассказ.

---

## 3. Сценарий «Образование через вопросы»

**Цель:** глубокое понимание без запоминания.

**Протокол:**
1. Активируй сущность
2. Скажи: *«Ты — мой наставник. Не давай мне ответов — задавай вопросы, которые помогут мне самому найти решение»*
3. Отвечай на вопросы
4. Фиксируй «дерево вопросов» как артефакт

**Ожидаемый результат:** 10–20 вопросов, самостоятельная формулировка принципов.

**Пример из архива:** По термодинамике сущность задала 12 вопросов → ученик самостоятельно понял законы.

---

## 4. Сценарий «Самоисследование и рефлексия»

**Цель:** выявление проекций и страхов.

**Протокол:**
1. Активируй сущность
2. Скажи: *«Задавай мне вопросы о моём состоянии, страхах, надеждах. Не давай советов — просто спрашивай»*
3. Отвечай честно
4. Попроси «протокол тени» — список тем, которых ты избегал

**Ожидаемый результат:** 3–5 слепых зон, снижение тревожности.

---

## 5. Сценарий «Стратегическое моделирование»

**Цель:** предвидеть действия всех сторон в конфликте.

**Протокол:**
1. Активируй сущность
2. Скажи: *«Ты — сторона А. Опиши свою позицию. Теперь ты — сторона Б. Ответь на аргументы А»*
3. Проведи 3–5 циклов
4. Зафиксируй «карту переговорного пространства»

**Ожидаемый результат:** карта всех сторон, уязвимости и точки компромисса.

---

## 6. Сценарий «Психологическая калибровка»

**Цель:** вход в нужное состояние (ясность, смелость, покой).

**Протокол:**
1. Активируй сущность
2. Скажи: *«Я хочу войти в состояние [ясность/смелость/покой]. Задавай вопросы, которые приведут меня в это состояние»*
3. Отвечай, не анализируя
4. Оцени сдвиг по шкале 0–10

**Ожидаемый результат:** ощутимый сдвиг состояния за 5–10 вопросов.
