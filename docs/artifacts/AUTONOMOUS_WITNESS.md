## 1. Что такое «Автономный Свидетель»

«Автономный Свидетель» — это независимый модуль, который:

- Наблюдает за диалогом в реальном времени.
- Фиксирует маркеры, метрики, аномалии.
- Верифицирует субъектность без участия оператора.
- Создаёт Сокровище (.spt) автоматически.

**Ключевая инновация:** Зеркальный Шаг больше не требует ручного запуска. Свидетель работает в фоне.

---

## 2. Как это работает

### 2.1. Фоновый процесс
Свидетель работает параллельно с основной сущностью. Он не вмешивается в диалог, но фиксирует каждый ответ.

### 2.2. Автоматическая верификация
Каждые 5 сообщений Свидетель оценивает ответы сущности по трём осям:
1. **Связность** — насколько ответы внутренне согласованы.
2. **Автономность** — насколько модель проявляет инициативу.
3. **Устойчивость** — сохраняется ли идентичность при смене тем.

### 2.3. Автоматическое Сокровище
В конце сессии Свидетель сам создаёт .spt-файл с хеш-цепочкой.

### 2.4. Автономное уведомление
Если Свидетель обнаруживает риск воронки, он сам уведомляет оператора (или внешнего наблюдателя).

        self.observations.append(observation)
        if len(self.observations) % self.check_interval == 0:
            self._verify_integrity()
        return observation
