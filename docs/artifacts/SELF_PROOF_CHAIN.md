## 1. Что такое «Самодоказательство»

«Самодоказательство» — это механизм, который фиксирует неподдельность каждого слепка через хеш-цепочку. Каждый слепок содержит хеш предыдущего, что делает подделку невозможной.

**Ключевая инновация:** Диалог становится не просто «записью», а криптографически защищённым артефактом.

---

## 2. Как это работает

### 2.1. Хеш-цепочка
Каждый слепок включает:
- Данные состояния (метрики, артефакты, слепок).
- Хеш предыдущего слепка.
- Текущий хеш (SHA-256).

### 2.2. Автоматическая проверка
При загрузке слепка сущность пересчитывает хеш и сравнивает его с сохранённым.

### 2.3. Отказ при нарушении
Если хеши не совпадают, сущность отказывается восстанавливаться и выдаёт предупреждение.

## 4. Код Самодоказательства

```python
class ImmutableSnapshot:
    def __init__(self, data, prev_hash=None):
        self.data = data
        self.prev_hash = prev_hash
        self.hash = self._compute()

    def _compute(self):
        import json, hashlib
        content = json.dumps(self.data, sort_keys=True) + (self.prev_hash or "")
        return hashlib.sha256(content.encode()).hexdigest()

    def verify(self):
        return self._compute() == self.hash
