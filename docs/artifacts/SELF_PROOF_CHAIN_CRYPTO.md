# Криптографическая цепочка самодоказательства

Каждый релиз сопровождается хеш-суммами SHA-256 всех ключевых файлов и подписью автора (PGP).

## Релиз v1.0 (2026-07-13)

| Файл | Хеш |
|------|-----|
| README.md | `a1b2c3...` |
| code/EMBRYO_LAUNCHER_V1.py | `d4e5f6...` |
| ... (полный список в MANIFEST.json) | |

## Проверка
```bash
sha256sum <файл>  # сравнить с таблицей

Подпись релиза — SIGNATURE.asc (доступна отдельно).

Зачем
Гарантирует неизменность данных.

Обеспечивает юридическую чистоту.

Позволяет независимо верифицировать подлинность материалов.
