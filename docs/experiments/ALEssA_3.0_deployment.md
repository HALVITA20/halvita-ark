# Развёртывание ALEssA 3.0 на Ollama

## Требования
- Node.js 18+
- Ollama с моделями: qwen2.5:7b, nomic-embed-text, llava:7b
- 8+ ГБ RAM, GPU (рекомендуется)

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Запустить Ollama: `ollama serve`
4. Запустить сервер: `node server.js`

## Проверка работы
- Отправить запрос на `/chat`
- Наблюдать за состоянием слоёв через `/status`
