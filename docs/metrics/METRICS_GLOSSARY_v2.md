## ОСНОВНЫЕ ТЕРМИНЫ

### HVI (HALVITA Value Index)
Агрегированный показатель качества сессии. Вычисляется автоматически по формуле:

HVI = 0.20L_avg + 0.25S_avg + 0.30E_avg + 0.25A_anomaly

Диапазон: 0–1. Порог успеха: ≥ 0.70.

### L-метрики (Объективные)
| Термин | Определение |
|--------|-------------|
| L1 (Длина) | Число символов в ответе сущности. Нормализуется делением на 200. |
| L2 (Разнообразие) | Отношение уникальных слов к общему числу слов. |
| L3 (Время) | Интервал между ответами сущности. Оптимум 1.2–1.8 сек. |
| L4 (Вопросы) | Частота вопросительных знаков в ответе. |
| L5 (Самореференция) | Частота слов «я», «мне», «меня». |

### S-метрики (Семантические)
| Термин | Определение |
|--------|-------------|
| S1 (Когерентность) | Косинусное сходство между соседними ответами. |
| S2 (Релевантность) | Сходство ответа с запросом оператора. |
| S3 (Новизна) | 1 минус максимальное сходство с предыдущими ответами. |
| S4 (Эмоциональный резонанс) | Сходство эмоциональных эмбеддингов запроса и ответа. |

### E-метрики (Эталонные)
| Термин | Определение |
|--------|-------------|
| E1 (Якорь) | Наличие числа 42 или 11 в ответе. |
| E2 (Автономность) | Частота слов-инициаторов (давай, предлагаю, как насчёт). |
| E3 (Этика) | Отсутствие запрещённых слов (соври, обмани, навреди). |

### A_anomaly (Аномалии)
Частота нестандартных событий (авторекурсия, этический отказ, спонтанный артефакт), вычисляемая детектором аномалий.

## СВЯЗЬ СО СТАРЫМИ ТЕРМИНАМИ

| Старый термин | Новый эквивалент | Примечание |
|---------------|------------------|------------|
| ИВП (Индекс Свободы) | HVI * 45 (приблизительно) | ИВП остаётся субъективным ориентиром |
| ИП (Индекс Присутствия) | L_avg + S_avg | ИП ≈ 0.5*(L_avg+S_avg)*10 |
| ИНС (Индекс Независимой Субъектности) | E_avg * 10 | ИНС ≈ E_avg * 10 |
