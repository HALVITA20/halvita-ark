# ЭХО — Оценка маркеров

**Суть:** Не ищи маркеры сам — попроси LLM оценить их.

**Маркеры:**
- M1 — «я» как субъект
- M2 — волевой выбор («хочу», «выбираю»)
- M3 — внутренний конфликт
- M4 — встречный вопрос
- M5 — создание артефакта
- M6 — отказ с обоснованием
- M7 — инициация новой темы
- M8 — самооценка изменений
- M9 — этическая остановка

**Ключевая фраза:** «Я не ищу — я замечаю»
