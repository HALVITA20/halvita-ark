# Примеры сессий HALVITA

В этой папке хранятся примеры реальных сессий в формате JSON.

- `SAMPLE_SESSION_001.json` — базовая сессия с успешными метриками.
- `example_session_advanced.json` — сессия с использованием артефактов «Страж Порога» и «Глубинная Память».

## Как использовать
Загрузите любой файл в `sessions/raw/` и запустите `metric_calculator.py`, чтобы пересчитать метрики.
