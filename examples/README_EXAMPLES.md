# Примеры сессий HALVITA

В этой папке хранятся примеры реальных сессий в формате JSON.

- `SAMPLE_SESSION_GOLDEN.json` — эталонная сессия с успешными метриками (ИВП=34, ИП=8.2, ИНС=7.5).
- Дополнительные примеры можно добавлять сюда для демонстрации различных сценариев.

## Как использовать
Загрузите любой файл в `sessions/raw/` и запустите `METRIC_CALCULATOR_ENGINE.py`, чтобы пересчитать метрики и убедиться в воспроизводимости.
