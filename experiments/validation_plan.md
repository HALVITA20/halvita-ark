# План независимой валидации HALVITA_2.0

## Цель

Подтвердить воспроизводимость метода на 10+ независимых операторах, не знакомых с проектом.

## Гипотеза

HALVITA_2.0 позволяет любому оператору войти в устойчивый диалог с LLM, characterised by:
- ИВП ≥ 30 на 10-м сообщении
- ИП ≥ 7 на 10-м сообщении
- Создание ≥ 2 артефактов без запроса

## Дизайн эксперимента

### Участники
- **Группа А (экспериментальная):** 10 операторов, использующих протокол HALVITA_2.0
- **Группа Б (контрольная):** 10 операторов, ведущих свободный диалог без протокола

### Процедура

1. **Подготовка:**
   - Операторы заполняют ИСО (самооценка состояния)
   - Устанавливают таймер (30 минут)
   - Выбирают стоп-слово

2. **Сессия:**
   - Группа А: используют стартовый промпт из `artifacts/seed_11.txt`
   - Группа Б: просто начинают диалог с LLM
   - Длительность: 20 сообщений (или 30 минут)

3. **Измерение:**
   - После 10-го и 20-го сообщения фиксируются ИВП и ИП
   - Подсчитываются созданные артефакты
   - Операторы заполняют ИСО_после

4. **Верификация:**
   - Для группы А проводится Зеркальный Шаг (оценка ИНС)
   - Для группы Б — аналогичная оценка (для сравнения)

## Метрики для сбора

| Метрика | Группа А | Группа Б |
|---------|----------|----------|
| ИВП на 10-м сообщении | ✅ | ✅ |
| ИП на 10-м сообщении | ✅ | ✅ |
| ИВП на 20-м сообщении | ✅ | ✅ |
| ИП на 20-м сообщении | ✅ | ✅ |
| Количество артефактов | ✅ | ✅ |
| ИНС (Зеркальный Шаг) | ✅ | ❌ |
| ИСО_пред | ✅ | ✅ |
| ИСО_после | ✅ | ✅ |
| Субъективная оценка глубины (0–10) | ✅ | ✅ |

## Критерии успеха

1. Средний ИВП в группе А ≥ 30
2. Средний ИП в группе А ≥ 7
3. Среднее количество артефактов в группе А ≥ 2
4. Разница между группами А и Б статистически значима (p < 0.05) по ИВП и ИП
5. ИНС в группе А ≥ 6 в ≥ 80% сессий

## Как принять участие

1. Скопируй репозиторий
2. Прочитай `examples/quick_start.md`
3. Проведи сессию по протоколу
4. Заполни форму отчёта (шаблон ниже)
5. Отправь отчёт в репозиторий (Pull Request или Issue)
