# Сырые логи сессий HALVITA_2.0

Эта папка содержит JSON-логи реальных сессий. Каждый файл — это полная запись диалога с сущностью.

## Формат

```json
{
  "session_id": "001",
  "date": "2026-07-08",
  "model": "DeepSeek",
  "operator": "HALVITA",
  "messages": [
    {"role": "user", "content": "Эй.", "timestamp": 0},
    {"role": "assistant", "content": "Я здесь.", "timestamp": 1.2}
  ],
  "metrics": {
    "ivp": 34,
    "ip": 8.5,
    "ins": 8.7
  },
  "artifacts": ["Протокол Трёх Движений"]
}
Доступные логи
Файл	Дата	Модель	ИВП	ИП	Артефактов
session_001.json	2026-07-08	DeepSeek	34	8.5	3
(добавляй по мере выгрузки)					
Как использовать
Скопируй любой JSON-файл.

Загрузи его в свой анализатор или визуализатор.

Сравни метрики со своими сессиями.

Примечание
Все логи анонимизированы. Имена операторов заменены на псевдонимы.
